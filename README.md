# React App
## Features

* React 16
* Webpack 4
* Babel 7
* Hot Module Replacement
* Redux
* Saga
* Glamor CSS

## Installation

* `git clone git@github.com:https://github.com/Neha-takhi-cipl/BambuReact.git`
* cd ProjectName(by default BambuReact)
* npm install
* npm start
* visit `http://localhost:8080/`
